# xXx Commanders · Kingshot Research Planner

A research priority advisor for [Kingshot](https://kingshot.net). Plan your Academy research order across Growth, Economy, and Battle trees — tailored to F2P, mid-spend, or whale playstyles.

**Live app:** [your-username.github.io/kingshot-advisor](https://your-username.github.io/kingshot-advisor/)

## Features

- 191 researches across all 3 trees, with official in-game names
- 12 languages (English, Hebrew, Russian, German, Spanish, French, Portuguese, Italian, Turkish, Chinese, Japanese, Korean)
- Spend-profile-aware priority queue (F2P / Mid / Whale)
- Why-this explanations for every research
- Search across all techs
- Export/Import progress backup
- Pin specific techs as top priority
- Dark mode by default
